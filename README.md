# Input

The program does not require any user input. When the application starts, React Native automatically loads the main screen. The program contains three predefined text values representing the student's name, degree program, and school abbreviation. These values are written directly in the source code and displayed without requiring any interaction from the user.

# Process

The application is built using React Native with Expo. A single View component acts as the main container, while three Text components display the student's name, degree program, and school. The StyleSheet component defines the visual appearance of the application by setting a yellow background, centering the content horizontally and vertically, and formatting the text with an appropriate font size and color. React Native renders these components to create the user interface when the application launches.

# Output

The application displays a simple mobile screen with a yellow background. At the center of the screen, the student's name, degree program (MSCS), and school abbreviation (STC) appear in black text on separate lines. The result is a clean and readable "Hello World" mobile application that meets all assignment requirements.
